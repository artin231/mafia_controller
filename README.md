# How to run this project
## 🪟 For Windows
```
python -m venv venv
venv/Scripts/activate
cd mafia
python manage.py runserver
```

---

## 🐧 For Gnu/Linux
```
python3 -m venv venv
source venv/bin/activate
cd mafia
python3 manage.py runserver
```
